## Setup
- Download Code from github
- Cmd cd ../project-root
- Cmd composer update
- Cmd php artisan optimize:clear
- Cmd php artisan key:generate
- setup Env File
- Cmd php artisan migrate --seed / php artisan migrate:refresh --seed
- Cmd php artisan serve
- Run SuccessFully
